# CpG

## About

- `finder`:
    Return the CpG island position based in Gardiner-Garden and Frommer or Takai and Jones criteria;

- `stats`:
    Return the sequence `length`, `CG count`, `GC_percent` and `Obs/Exp` ratio;

- `profiler`:
    Return the array of a given stats such `CG count`, `GC_percent` and `Obs/Exp` in a sliding window.
